Web Application Programming and Hacking

Course and Hands-On Projects

Instructor: Dr. Phu Phung

Course Focus

WAPH connects front-end development practice with web security awareness.

The course covers client-side web programming, HTTP-based application behavior, public API usage, browser storage, JavaScript events, and common security concepts used when evaluating or defending web applications.

This project applies those topics through a professional profile website hosted on GitHub Pages, with Bootstrap styling, JavaScript interactions, jQuery usage, API integration, cookies, and a page tracker.

Hands-On Work

Lab 2 JavaScript Features

Digital clock, analog clock, show/hide email behavior, and an additional theme toggle interaction.

Public Web APIs

JokeAPI provides rotating text content, while Dog CEO API provides image content from a public graphics API.

Client-Side Persistence

JavaScript cookies remember whether the visitor is new and store the previous visit date and time.

Deployment Target
